# Sales Performance Analysis Using SQL
## Overview 
This project analyzes sales performance date using SQL to identify revenue trends, regional performance, product category insights and discount patterns.

## Tools Used
- SQL
- MySQL
- Data Analysis
- KPI Reporting
- Business Intelligence

## Opjectives
- Analyse sales performance across regions
- Identify top-performing product categories
- Evaluate discount impact on revenue
- Generate business KPIs

## SQL Concepts Applied
- Aggregate Functions
- GROUP BY
- ORDER BY
- SUM()
- AVG()
- ROUND()

## Key Insights
- Total revenue exceeded 5M+
- North region generated the highest revenue, indicating strong market demand compared to other regions
- Average discount rate reached 15%

## Business Impact
this analysis helps identify high-performing regions and optimize discount strategies to improve profitability
