# Sales Performance Analysis Using SQL
## Overview 
This project analyzes sales performance date using SQL to identify revenue trends, regional performance, product category insights and discount patterns.

## Tools Used
_ SQL
_ MySQL
_ Data Analysis
_ KPI Reporting
_ Business Intelligence

## Opjectives
_ Analyse sales performance across regions
_ Identify top-performing product categories
_ Evaluate discount impact on revenue
_ Generate business KPIs

## SQL Concepts Applied
_ Aggregate Functions
_ GROUP BY
_ ORDER BY
_ SUM()
_ AVG()
_ ROUND()

## Key Insights
_ Total revenue exceeded 5M+
_ North region generated the highest revenue, indicating strong market demand compared to other regions
_ Average discount rate reached 15%

## Business Impact
this analysis helps identify high-performing regions and optimize discount strategies to improve profitability
